-------project in development-------

Visualizing real time global fire data on the world map.
Including separate page with graphs based on historical data and current Twitter news feed

Data sources:
 - NASA Active Fire Data
 - NASA Historic Fire Data
 - Twitter API
 - NASA Earth Images API
 
 Technologies used:
 Flask app, JavaScript, Python, SQL, Postgress, noSQL, Mapbox, Leaflet, Plotly, html, CSS, Bootstrap
